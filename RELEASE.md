# Release v1.0.0 — Grok Build Z.ai Edition

## Highlights

- First stable release of Grok Build Z.ai Edition
- 8 Z.ai GLM models fully configured
- Automated installation script
- Comprehensive test suite
- Complete documentation

## What's Included

| Component | Description |
|-----------|-------------|
| `config/config.toml` | Pre-configured Grok Build CLI config with all 8 Z.ai models |
| `scripts/install.sh` | One-command installation script |
| `scripts/configure.sh` | Configuration verification tool |
| `tests/` | Python test suite (config, models, connectivity) |
| `docs/` | Full documentation (setup, models, FAQ, troubleshooting) |

## Supported Models

- GLM-5.1 (default)
- GLM-5
- GLM-5 Turbo
- GLM-5V Turbo
- GLM-4.7
- GLM-4.7 Flash
- GLM-4.6V
- GLM-5.2

## How to Install

```bash
git clone https://github.com/roman-ryzenadvanced/grok-build-zai-edition.git
cd grok-build-zai-edition
chmod +x scripts/install.sh
./scripts/install.sh
```
